Navodila:

Jar: json-java.jar naj bo v isti mapi, treba ga je dodati v referenced libraries

Compile: javac -cp json-java.jar *.java

Start server: java -cp json-java.jar ChatServer.java

Start client: java -cp json-java.jar ChatClient.java

Za pošiljanje vsem (public) napišete le sporočilo in pošljete

Za pošiljanje določenemu uporabniku napišete / imeuporabnika ipnaslov port in za tem sporočilo

Primer: / pino 127.0.0.1 52127 Dober dan
